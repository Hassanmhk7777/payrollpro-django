Questions Fréquentes

Comment ajouter un nouvel employé ?

Pour ajouter un nouvel employé :

  1. Allez dans "Liste des Employés"
  2. Cliquez sur "Ajouter un employé"
  3. Remplissez tous les champs obligatoires
  4. Sauvegardez les informations
Comment calculer la paie mensuelle ?

Le calcul de paie se fait automatiquement :

  1. Allez dans "Calcul de Paie"
  2. Sélectionnez la période
  3. Choisissez les employés
  4. Lancez le calcul automatique
Comment gérer les permissions utilisateur ?

La gestion des permissions :

  • Admin : Accès complet au système
  • RH : Gestion employés et paie
  • Employé : Consultation de ses informations
